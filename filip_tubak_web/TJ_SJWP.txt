IZVOĐENJE VJEŽBE

Pri traženju odgovora koristite se internet tražilicom.

1. WEB preglednici
 A. Što rade i za što se koriste web preglednici, čemu sve veću važnost pridaju suvremeni
preglednici? 
-Web preglednik je softverski program koji korisniku omogućuje lociranje, pristup i prikaz web stranica.
 Web-preglednik upravlja HTTP aktivnošću između klijenta i poslužitelja koji je osnova korištenja World Wide Weba.
 URL-ovi su prometne upute za web preglednik, 
 a preglednik koristi IP adrese i druge alate za uspostavljanje tih veza.
 Suvremeni preglednici sve veću važnost dodaju sigurnosti i pristupačnosti.
 B. Što su plug-in-ovi, što omogućuju i kako se dodaju pregledniku? Navedite primjere.
-Pluginovi su proširenja i dodaci koji čine web preglednik znatno korisnijim, ali mogu i usporiti njegov rad.
 Svaki web preglednik ima stranicu sa koje se mogu preuzeti plug-inovi
 Neki od najpoznatijih plug-inova su: Adobe Acrobat, Adobe Flash, Java, Silverlight
 C. Koji su koraci u radu preglednika nakon upisa URL-a?
-Prvo će se stranica ping-ati. Ako postoji učitat će se, a ako dođe do greške ta greška će se ispisati na ekranu.
 Ako stranica ne postoji onda će se ispisati da stranica ne postoji.
 D. Skicirajte shematski prikaz preglednika

 E. Što je cross browser testing i zašto je važno?
-Cross Browser testiranje vrsta je nefunkcionalnog testiranja koje vam omogućuje da provjerite 
 radi li vaše web mjesto kako je predviđeno kada mu se pristupa putem: 
 Različite kombinacije preglednika i OS-a, različitim uređajima i pomoćnim alatima.
 Cross browser testing je važan za osiguranje da bilo tko s bilo kojim WEB preglednikom 
 može pristupiti WEB stranici.


2. WEB Pretraživači

 A. Objasnite općenito kako rade tražilice. 
Odaberite neku od predloženih i objasnite kako rade njihovi algoritmi (Google, Bing, Youtube, DuckDuckGo i sl.)

 B. Objasnite zašto je važno znati kako rade web pretraživači.
-Važno nam je znati kako rade WEB pretraživači da možemo znati koji web pretraživač bi bio najbolji i najsigurniji .
 C. Na koje se načine ostvaruje upotrebljivost tražilice?
-Upotrebljivost tražilice ostvaruje se dobrim dizajnom, rasporedom, brzinom, pristupačnosti i dobrim rezultatima.
 D. Što podrazumijeva personalizacija tražilica?
-Personalizirano pretraživanje značajka je koja prilagođava rezultate tražilice jedinstvenim preferencijama,
 interesima i prošlim ponašanjima pojedinca.
 E. Objasnite SEO (Kako funkcionira i zašto je važno?)
-SEO (Search Engine Optimization) je proces dizajniranja, pisanja, kodiranja, programiranja i skriptiranja 
 web stranice u cilju boljeg rangiranja na pretraživačima.
-Bez dobre SEO optimizacije, postoji rizik da se izgube potencijalni kupci nad konkurentima jer se njihove 
 web stranice u rezultatima pretrage prikazuju na višem nivou.
